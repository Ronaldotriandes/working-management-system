## Installation

Use the package manager [npm](https://www.npmjs.com/) to install 
Or use with yarn to install 

FOR BACKEND

## go to folder backend
```bash
cd backend
```

## Install npm
```bash
npm install --save
```

## Set ENV
```bash
DB_NAME=work-management
DB_USER=
DB_PASSWORD=
DB_HOST=localhost
PORT=8080
JWT_SECRET=
```

## Run this app with npm
```bash
npm run dev
```

## FOR FRONTEND

## go to folder frontend
